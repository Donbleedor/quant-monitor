## 总结

当前版本不是简单的“整体缩放不准”，而是四类问题叠加：

1. 部分透明资源仍有粉色/紫色残留。
2. 资源画布尺寸与资源实际可见区差异过大。
3. 完整面板、按钮壳和状态图被重复叠加或使用方式错误。
4. 外层虽然固定为 9:20，但内部仍混用固定像素、`vw`、百分比和硬上限，无法真正等比适配。

建议先完成 P0 返修，再做性能、可访问性和未截图状态的 P1 验收。
