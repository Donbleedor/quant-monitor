# 提交给程序说明

这个目录下有两个包：

## 1. 静态美术主包_PNG_必须接入

这是本次必须提交和接入的主资源包。

程序应从这里开始看：

```text
静态美术主包_PNG_必须接入/README_FOR_PROGRAMMER.md
```

资源内容：

- 游玩界面 PNG
- 主菜单 PNG
- 弹窗 PNG
- 验收报告
- 接入注意事项

## 2. 可选Spine动效_Tidy版

这是可选动效增强包，不是 P0 必接内容。

程序应从这里开始看：

```text
可选Spine动效_Tidy版/README_FOR_PROGRAMMER.md
```

建议：

- 先完成静态 PNG 换皮。
- 再评估是否接 Spine 动效。
- 首推测试 `gameplay_box_broken_eliminate`。

## 不要提交

根目录下的 `99_可能可删除_过程归档` 不建议交给程序作为正式接入包。

里面主要是旧包、生产过程、验收过程和临时工作区。
